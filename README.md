The frontend of Dueet Live.

## Setup guide

1. Clone the repository
2. Run

   ```bash
   cp .env.example .env
   ```

   and fill in the environment variables based on how the backend is set up.

3. Install dependencies

   ```bash
   yarn install
   ```

4. Run the app in development mode

   ```bash
   yarn start
   ```
